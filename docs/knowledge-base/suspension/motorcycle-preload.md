# Motorcycle preload

Preload sets the starting position of the spring, which controls static sag (how much the bike settles under its own weight) and rider sag (how much it settles with the rider aboard).

## What preload does

Preload does not change spring rate. It changes where in the travel the suspension starts. Adding preload raises the ride height and reduces sag. Removing preload lowers the ride height and increases sag.

## Target sag ranges

Typical trackday targets (confirm with your manufacturer and tire supplier):

- Front rider sag: 30-35 mm on sportbikes.
- Rear rider sag: 25-30 mm on sportbikes.
- Static sag (no rider): front 15-20 mm, rear 5-10 mm.

If rider sag is in range but static sag is near zero, the spring is too soft for the rider weight (preload cannot fix this). If rider sag is in range but static sag is large, the spring may be too stiff.

## Common mistakes

- Using preload to fix a dive/squat symptom. Preload changes ride height, not damping response. Fix dive with compression; fix wallow with rebound.
- Adding preload to "firm up" a too-soft spring. It changes geometry, not stiffness. The spring still bottoms at the same force.
- Matching front and rear preload blindly. Front and rear are independent; tune each to its own sag target.

## Interactions

Raising front preload steepens rake, quickens steering, reduces stability. Raising rear preload increases rear ride height, quickens steering, can add wheelie tendency. Make one change at a time and measure sag again before riding.
